# F-ROS 快速參考卡

**博士級 × 期刊級 × OpenClaw 可執行 × SCI Agent 自動化**

---

## 🎯 四套系統速查

| 系統 | 用途 | 核心功能 |
|------|------|----------|
| **PhD Deep Mode** | 博士研究 | 5問題、主題歸納、矛盾分析、缺口識別 |
| **Reviewer Killer** | 期刊投稿 | 方法學審查、弱點找尋、防禦方案 |
| **Research Engine** | 系統整合 | YAML配置、子代理連結 |
| **SCI Agent** | 終極分析 | 6步流程、全面評估、信心水平 |

---

## 🚀 快速使用

### PhD Deep Mode

```markdown
任務：分析 [論文/主題]

輸出：
1. 5個核心問題
2. 主題歸納表
3. 矛盾分析
4. 研究缺口
5. 章節建議
```

### Reviewer Killer Mode

```markdown
任務：預審 [論文]

輸出：
1. 主要發現
2. 方法優勢/缺陷
3. 潛在質疑點
4. 防禦方案
```

## 📋 Prompt 模板庫

### Template 1：PhD Deep

```markdown
你是博士論文研究助理。

任務：分析 [插入內容]

強制步驟：
1. 生成5個核心問題
2. 文獻主題歸納（5–10個）
3. 矛盾分析
4. 研究缺口
5. 章節建議

規範：
- 所有主張必須附證據
- 分離事實與推論
- 承認資料不足處
```

### Template 2：Reviewer Killer

```markdown
你是期刊審稿前內部評估專家。

任務：預審 [插入論文]

輸出：
【主要發現】
【方法優勢】
【方法缺陷】
【統計完整性檢查】
【潛在審稿質疑點】
【如何預先防禦】
```

### Template 3：SCI Agent

```markdown
你是資深跨領域科學研究顧問。

任務：對 [主題] 進行高階研究分析。

流程：
STEP 1：5個核心研究問題
STEP 2：主流理論與實證立場
STEP 3：關鍵矛盾與衝突證據
STEP 4：未解決研究缺口
STEP 5：方法學嚴謹度評估
STEP 6：可發表等級改進建議

輸出結構：
I. 概覽
II. 證據支持之主張
III. 方法學批判
IV. 矛盾與限制
V. 可發展新研究方向
VI. 信心水平評估
```
