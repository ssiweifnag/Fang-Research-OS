# Contributing to Fang Research OS

Thank you for your interest in contributing to Fang Research OS!

## How to Contribute

### 1. Fork the Repository

Click the "Fork" button at the top right of this page.

### 2. Clone Your Fork

```bash
git clone https://github.com/YOUR_USERNAME/Fang-Research-OS.git
cd Fang-Research-OS
```

### 3. Create a Branch

```bash
git checkout -b feature/AmazingFeature
```

### 4. Make Changes

- Follow the existing code style
- Add tests if applicable
- Update documentation

### 5. Commit Your Changes

```bash
git commit -m 'Add some AmazingFeature'
```

### 6. Push to GitHub

```bash
git push origin feature/AmazingFeature
```

### 7. Open a Pull Request

Go to the original repository and click "New Pull Request".

## Contribution Guidelines

### Code Style
- Use 4 spaces for indentation
- Follow PEP 8 for Python
- Use descriptive variable names

### Documentation
- Update README.md for new features
- Add comments for complex logic
- Use English for code comments

### Testing
- Write tests for new functionality
- Ensure all tests pass before submitting

### Reporting Issues
- Use the GitHub Issues tab
- Describe the problem clearly
- Include reproduction steps if applicable

## Areas for Contribution

| Area | Description | Priority |
|------|-------------|----------|
| Automation Scripts | Add more workflow scripts | High |
| Prompt Templates | Create new prompt variants | High |
| Test Suite | Expand test coverage | Medium |
| Documentation | Improve guides and examples | Medium |
| Docker | Add Docker support | Low |
